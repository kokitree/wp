== Description ==
Iriska is a Blog WordPress Theme with many settings. In the Theme there is a widget with the latest posts, several widgets areas. Theme also has many areas for widgets for pages, posts, archives, footer and some areas can be customized in width. You can connect any font from Google Font. Customize all colors and layout. Qualitatively made up layout that looks great on all screen resolutions.

== Installation ==
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's iriska.zip file. Click Install Now.
3. Click Activate to use your new theme right away.
